## **«Код: Перезагрузка» (Code: Reboot)**

### **Жанр**
Обучающий платформер с элементами программирования и мета-игрой.

### **Сюжет**
Вы — искусственный интеллект, загруженный в повреждённую систему. Чтобы выбраться, необходимо восстановить управление, редактируя код самой системы. Каждый уровень — это головоломка, где игрок взаимодействует с окружением и изменяет переменные через терминал.

### **Цель игры**
На первом уровне: включить генератор, затем через терминал изменить значение переменной `exitDoorLocked` с `true` на `false`, чтобы открыть выход и перейти на следующий уровень.

### **Основные механики**
- **Свободное перемещение** по комнате с видом сверху (WASD / стрелки).
- **Взаимодействие** с объектами (E): генератор включает/выключает питание, терминал открывает редактор кода, выход проверяет статус двери.
- **Редактор кода** – отдельное окно, где игрок выбирает переменную из списка (стрелками) и вручную вводит новое значение. Поддерживаются типы `bool` и `int`, с валидацией ввода.
- **Системные переменные** влияют на игровой мир: без энергии терминал не работает, а выход открывается только после смены `exitDoorLocked`.
- **Управление камерой** с центрированием на игроке.
- **Полноэкранный режим** (F4) и выход по удержанию ESC (2 секунды).

### **Технологии**
- **Язык:** C# (чистый код, без Unity)
- **Фреймворк:** WPF (.NET Framework) для графики и интерфейса
- **Сборка:** Visual Studio 2022

### **Особенности реализации**
- Игровой цикл построен на `CompositionTarget.Rendering` для плавного движения.
- Все объекты (стены, генератор, терминал, выход) являются препятствиями с коллизиями.
- Логика терминала полностью реализована на C# без внешних библиотек.
- Редактор кода использует диалоговое окно с поддержкой выбора переменной и ручного ввода.

### **Чему учит игра**
- Пониманию переменных и их типов (bool, int)
- Логике взаимодействия в игровом мире
- Основам работы с кодом в безопасной среде
